We value your privacy. To add or update your callsign details in the directory:

1.  Click the **+ Register / Update** button on the website.
2.  A popup will provide a template.
3.  **Email your details** to: `9m2pju@hamradio.my`

### Required Format
```text
Callsign: [Your Callsign]
Name: [Your Name]
Location: [State/City]
Email: [Optional]
Phone: [Optional - Publicly Visible]
Address: [Optional - Publicly Visible]
```
